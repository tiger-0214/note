# 过程分析汇总（针对数据分析）
## 一、论文了解初步
1、下载论文PDF原文  
2、分析论文内容：
  将文章内容分区块理解，大致分成:  
  "数据来源（dataset、data availablility、data...）"  
  "数据预处理过程（preprocessing data、模态、...）"  
  "输入输出内容以及分析对象（）"  
  "训练过程（training、...）"  
  以及高频词标记
3、根据区块内容，找到数据集来源（某个下载地址链接）&源代码地址（github）  
4、理解整个项目的工作内容（包括输入输出，项目目的，数据预处理过程（方法），模型训练过程（方法），）
